Informasi Kontak

Kosongkan jika ingin peta otomatis dibuat dari Alamat Lengkap di atas. Untuk pin lokasi yang presisi, buka lokasi di Google Maps → tombol Bagikan → tab "Sematkan peta" → salin URL di dalam atribut src lalu tempel di sini.

Konfigurasi Backend

Deploy file Code.gs sebagai Web App di Google Apps Script, tempel URL-nya di bawah. Pastikan juga GALERI_FOLDER_ID di Code.gs sudah diisi ID folder Google Drive yang akan menyimpan foto galeri.

Ganti Password Admin

Password lama wajib diverifikasi server sebelum diganti. Password tersimpan sebagai hash SHA-256 di sheet Admin.
